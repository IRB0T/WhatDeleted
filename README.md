# 🗑️ WhatDeleted — EnCase Deletion Analysis Made Simple

Ever wondered if deletion analysis using EnCase could be reduced to just a few clicks?

**WhatDeleted** is a lightweight DFIR utility that helps investigators quickly analyze **EnCase File Listings** and filter investigative noise to focus only on meaningful deleted artifacts.

Instead of manually sorting through millions of rows, WhatDeleted provides an interactive interface to rapidly narrow down deletion activity.

---

## 🚀 Overview

WhatDeleted simplifies deletion analysis by:

✅ Parsing large EnCase file listings  
✅ Filtering deleted artifacts instantly  
✅ Reducing investigative noise  
✅ Providing extension-based insights  
✅ Exporting refined datasets for reporting

The tool is designed for **DFIR analysts**, **incident responders**, and **digital forensic investigators** who frequently work with EnCase exports.

---

## 🎯 Problem Statement

Traditional deletion analysis workflow:

- Export EnCase file listing

## 💡 Solution — WhatDeleted

With WhatDeleted:

> Deletion analysis becomes a **few-click workflow**.

Upload → Filter → Analyze → Export.

---

## 🧰 Features

### 📂 EnCase File Listing Analysis
- Supports large EnCase file listings
- Reads millions of rows efficiently
- Browser-based interface

### 🔎 Dataset Filters
Filter investigation scope instantly:

- **Is Deleted**
  - Shows entries where Deleted = YES

- **Recycle Bin**
  - Focus only on recycle bin activity

- **AppData Deleted**
  - Include deleted AppData artifacts when required

---

### 📊 Extension Scope Analysis

Choose how extensions are analyzed:

- All extensions from deleted entries
- Top 25 extensions by count
- DFIR-relevant extensions only

Generate extension statistics with one click.

---

### 📋 Interactive Filtering
- Extension filtering
- Date-based filtering
- Dataset refinement
- Investigation-focused narrowing

---

### 📤 Export Options
- Export filtered results
- Download refined dataset as CSV
- Ready for reporting or timeline analysis

---

## 🖥️ How It Works

1. Launch `WhatDeleted.exe`
2. Browser opens automatically
3. Upload EnCase file listing
4. Click **Begin Analysis**
5. Apply dataset filters
6. Generate extension statistics
7. Refine results
8. Export filtered dataset
