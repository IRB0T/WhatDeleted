# 🗑️ WhatDeleted — EnCase Deletion Analysis Made Simple

> Turn EnCase deletion analysis into a **few-click workflow**.

**WhatDeleted** is a lightweight DFIR utility that helps investigators quickly analyze **EnCase File Listings** and eliminate investigative noise to focus only on meaningful deleted artifacts.

Instead of manually filtering millions of rows, WhatDeleted provides an interactive browser-based interface for rapid deletion analysis.

---

## 🚀 Features

- ✅ Parse large EnCase file listings efficiently
- ✅ One-click deletion dataset filtering
- ✅ Recycle Bin focused analysis
- ✅ Optional AppData artifact inclusion
- ✅ Extension-based investigation workflow
- ✅ Interactive filtering and refinement
- ✅ Export filtered results to CSV
- ✅ Standalone executable (No Python required)

---

## 🧰 Workflow

1. Launch WhatDeleted.exe
2. Browser opens automatically
3. Upload EnCase File Listing (TXT export)
4. Apply dataset filters
5. Generate extension statistics
6. Narrow down results
7. Export investigation-ready dataset

---

## ⚙️ Requirements
- Windows 10 / 11
- EnCase File Listing (TXT format)
- EnCase File Listing should have date format DD/MM/YY (13/11/22). Make sure to change date format before exporting the listing from the EnCase
- No additional dependencies or Python installation required.
