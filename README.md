# 🗑️ WhatDeleted — EnCase Deletion Analysis Made Simple

> Turn EnCase deletion analysis into a **few-click workflow**.

**WhatDeleted** is a lightweight DFIR utility that helps investigators quickly analyze **EnCase File Listings** and eliminate investigative noise to focus only on meaningful deleted artifacts.

Instead of manually filtering millions of rows, WhatDeleted provides an interactive browser-based interface for rapid deletion analysis.

## 🚀 Why WhatDeleted?

Traditional workflow using EnCase file listings:

- Export file listing
- Open massive TXT/CSV files
- Apply manual filters
- Identify relevant extensions
- Re-filter repeatedly
- Export investigation datasets

⏱️ Time consuming  
🧠 Analyst fatigue  
📉 Inefficient analysis
